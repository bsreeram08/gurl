# Skills for Gurl CLI

Skills for using the Gurl API client in AI agents.

## Available Skills

### gurl-cli
AI agent skill for using Gurl CLI commands.

```bash
npx skills add bsreeram08/gurl@gurl-cli
```

Features:
- All commands (save, run, list, import, etc.)
- Agent integration examples (Bash, Python, Node.js)
- Common workflows (Stripe, GitHub, CRUD)
- Configuration guide

**Directory structure:**
```
gurl-cli/
├── SKILL.md          # Main skill file
├── examples/         # Quick examples
└── workflows/       # Common API workflows
```
