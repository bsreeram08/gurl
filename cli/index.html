Gurl CLI Reference#

Gurl is a command-line HTTP client for saving, organizing, and executing curl requests.

Commands#

Core#

CommandDescription
saveSave a curl request with a name
runExecute a saved request
listList saved requests
authDiscover supported authentication types
editEdit a saved request
deleteRemove a saved request
renameRename a saved request

CommandDescription

Quick Start#

# Save your first request
gurl save "my-request" https://api.example.com/users

# Run it
gurl run "my-request"

# List all saved requests
gurl list

# Edit in your default editor
gurl edit "my-request"

# See available auth handlers
gurl auth list

gurl auth

gurl auth Discover the auth handlers built into gurl and see the parameters each handler accepts. Usage gurl auth list gurl auth info <type> Description Use gurl auth when you need the exact value for --auth or the supported --auth-param key=value pairs for a handler. --auth-param can be repeated on gurl save and gurl edit. gurl run uses the auth config already saved on the request. Auth parameters are template-substituted during execution, after the request URL, headers, and body are prepared and before the auth handler updates the outgoing request. ...

gurl delete

gurl delete Remove a saved request from gurl’s storage. Usage gurl delete [name] Description The delete command permanently removes a saved request. This action cannot be undone. Flags None. Aliases rm del d Examples Delete a request gurl delete "old-request" Removes the request named “old-request”. See also gurl save - Save a new request gurl list - List saved requests gurl rename - Rename a request
