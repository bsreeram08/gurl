# gurl tui

Launch the interactive terminal user interface for gurl.

## Usage

```bash
gurl tui
```

## Description

The `tui` command opens an interactive terminal interface where you can browse, search, edit, and execute requests. The TUI provides a visual way to work with your saved requests.

## Flags

None.

## Aliases

- `ui`

## Examples

### Launch TUI

```bash
gurl tui
```

Opens the interactive terminal interface.
